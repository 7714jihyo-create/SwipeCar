# SwipeCar
## 🏎️ 스와이프 조작 미니카 게임

유니티(Unity) 2D 환경에서 개발한 캐주얼 하이퍼코퍼레이티브 미니카 조작 게임입니다. 사용자의 마우스/터치 드래그(Swipe) 거리와 방향을 실시간으로 감지하여 자동차를 가속시키고, 마찰력에 의한 자연스러운 감속을 통해 목표 지점인 깃발 근처에 정확히 멈추는 것을 목표로 합니다.

<br>

## 🎮 주요 기능 (Key Features)

* **정밀한 스와이프 입력 처리**: 클릭 시작점(`GetMouseButtonDown`)과 뗀 지점(`GetMouseButtonUp`)의 X축 변화량(Delta)을 계산하여 드래그 길이에 비례한 직관적인 속도 제어를 지원합니다.
* **실시간 목표 가이드 시스템**: 드래그가 시작되는 순간, 현재 자동차의 위치로 깃발(`flag`) 오브젝트를 실시간 이동시켜 플레이어에게 시각적인 출발 기준점을 제공합니다.
* **물리 마찰력 시뮬레이션**: 유니티 내장 물리 엔진 대신 커스텀 감속 비율(`deceleration`) 연산을 적용하여, 매 프레임 부드럽고 예측 가능한 미끄러짐과 정지 효과를 구현했습니다.
* **오디오 피드백 루프**: 조작이 완료되어 자동차가 출발하는 타이밍에 맞춰 `AudioSource` 컴포넌트를 통해 즉각적인 엔진/주행 효과음을 재생하여 몰입감을 높였습니다.

<br>

## 🛠️ 기술 스택 (Tech Stack)

* **Engine**: Unity 2022.3.62f1
* **Language**: C#
* **Physics**: Transform-based Movement, Custom Deceleration (마찰력 알고리즘)
* **Audio**: AudioSource Component (One-shot SFX)
